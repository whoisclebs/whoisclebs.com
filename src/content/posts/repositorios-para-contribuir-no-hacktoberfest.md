O Hacktoberfest chegou e estamos a todo gás.

Esses primeiros dias, para mim, foram uma mistura de emoções: troca de conhecimento, contato com pessoas novas e muita contribuição com a comunidade. Mas também percebi um problema comum: por mais divertido e acolhedor que o Hacktoberfest seja, quem ainda não teve contato com desenvolvimento open source pode ficar um pouco assustado ou perdido com tanta informação, tantos repositórios e tantas formas de começar.

Então veio a pergunta:

## Como resolvemos isso?

Ainda não tenho uma resposta simples ou definitiva, mas uma boa forma de começar é com uma lista de repositórios que podem servir como ponto de entrada.

## Octobear.org

Octobear é uma comunidade open source amigável para contribuidores iniciantes. O repositório usa Next.js e Tailwind CSS para reunir recursos e oportunidades para pessoas desenvolvedoras.

- Repositório: https://github.com/OctobearHub/octobear.org

## Coffee Tasks

Coffee Tasks tem como objetivo criar uma experiência simples de lista de tarefas com uma pegada divertida. É um projeto interessante para quem quer contribuir com React, Vite e Tailwind CSS.

- Repositório: https://github.com/SauveJeanLuc/coffee-tasks

## URL Shortener

Como o nome sugere, URL Shortener é um encurtador de links simples, útil e eficaz para aprender. O projeto usa Pug e Express.

- Repositório: https://github.com/origranot/url-shortner

## Danet

Danet é um projeto um pouco mais avançado: um framework web para Deno fortemente inspirado no NestJS. Apesar de parecer mais complexo à primeira vista, a equipe documenta bem o projeto e organiza issues que podem ajudar pessoas novatas a contribuir.

- Repositório: https://github.com/Savory/Danet

## freeCodeCamp

freeCodeCamp é uma organização sem fins lucrativos com uma plataforma interativa de aprendizagem, fórum, publicações e comunidades locais. Contribuir com o projeto é uma forma de impactar diretamente ou indiretamente milhares de pessoas desenvolvedoras pelo mundo.

- Repositório: https://github.com/freeCodeCamp/freeCodeCamp

## LinkFree

LinkFree é uma alternativa open source ao Linktree. O projeto tem uma comunidade ativa, muitos mantenedores e uma boa estrutura para auxiliar pessoas contribuindo pela primeira vez.

- Repositório: https://github.com/EddieHubCommunity/LinkFree

## Como contribuir

Você pode contribuir de várias formas:

- adicionando novos recursos;
- resolvendo issues existentes;
- criando novas issues com descrições claras;
- melhorando documentação;
- testando funcionalidades;
- sugerindo melhorias.

À medida que eu encontrar novos repositórios interessantes, pretendo atualizar essa lista e trazer mais conteúdo sobre open source.

Obrigado pela leitura e até a próxima.
