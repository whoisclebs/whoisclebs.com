Hacktoberfest has arrived and we're going full steam.

These first few days, for me, were a mix of emotions: knowledge sharing, contact with new people, and lots of community contribution. But I also noticed a common problem: as fun and welcoming as Hacktoberfest is, people who haven't had contact with open source development yet can get a little scared or lost with so much information, so many repositories, and so many ways to start.

So the question came up:

## How do we solve this?

I don't have a simple or definitive answer yet, but a good way to start is with a list of repositories that can serve as an entry point.

## Octobear.org

Octobear is an open source community friendly to beginner contributors. The repository uses Next.js and Tailwind CSS to bring together resources and opportunities for developers.

- Repository: https://github.com/OctobearHub/octobear.org

## Coffee Tasks

Coffee Tasks aims to create a simple to-do list experience with a fun twist. It's an interesting project for those who want to contribute with React, Vite, and Tailwind CSS.

- Repository: https://github.com/SauveJeanLuc/coffee-tasks

## URL Shortener

As the name suggests, URL Shortener is a simple, useful, and effective link shortener for learning. The project uses Pug and Express.

- Repository: https://github.com/origranot/url-shortner

## Danet

Danet is a slightly more advanced project: a web framework for Deno heavily inspired by NestJS. Although it may seem more complex at first glance, the team documents the project well and organizes issues that can help newcomers contribute.

- Repository: https://github.com/Savory/Danet

## freeCodeCamp

freeCodeCamp is a nonprofit organization with an interactive learning platform, forum, publications, and local communities. Contributing to the project is a way to directly or indirectly impact thousands of developers around the world.

- Repository: https://github.com/freeCodeCamp/freeCodeCamp

## LinkFree

LinkFree is an open source alternative to Linktree. The project has an active community, many maintainers, and a good structure to help people contributing for the first time.

- Repository: https://github.com/EddieHubCommunity/LinkFree

## How to contribute

You can contribute in several ways:

- adding new features;
- solving existing issues;
- creating new issues with clear descriptions;
- improving documentation;
- testing functionality;
- suggesting improvements.

As I find new interesting repositories, I plan to update this list and bring more open source content.

Thanks for reading and see you next time.
